Hugo is a fast static site generator written in Go. It is designed for speed and flexibility, and is well suited for building blogs, documentation sites, and digital gardens.

## Key features

- Extremely fast build times
- Flexible content model
- Built-in support for taxonomies, menus, and i18n
- Template system based on Go's `html/template` package
- Live reload during development
