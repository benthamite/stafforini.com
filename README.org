#+title: stafforini.com

* Skills

** gsc-indexing-triage

The project-local =gsc-indexing-triage= skill handles Google Search Console
Page indexing alerts for =stafforini.com=. Read-only triage includes opening
Search Console in a browser to inspect failed examples and checking the
affected URLs locally or live.

Deployment, clicking Search Console validation buttons, and archiving Gmail
messages remain externally visible actions and require explicit authorization
in the current invocation.

* Note Export

The note exporter resolves =id:= links through the generated ID maps.
Links to unpublished targets become plain text, links to published
notes become =/notes/slug/= URLs, and links to headings inside the
same note become local fragment links such as =#copycat-delays=.

** SA LP Refresh

=scripts/sa-lp-refresh.sh= evaluates every dynamic Babel block used by the
Situational Awareness LP note, including the AIS comparison block, before
exporting and committing the generated static chart/calculator assets.
